# Apple Homepage Clone

A responsive Apple-inspired homepage with smooth animations and interactive dropdown menus.

## Technologies

- HTML5
- CSS3
- JavaScript
- Tailwind CSS

## Features

- Responsive navigation with dropdown menus
- Smooth animations and transitions
- Mobile-friendly design
- Interactive components

## Getting Started

Simply open `index.html` in your web browser.

```bash
start index.html
```

```
├── index.html
```
